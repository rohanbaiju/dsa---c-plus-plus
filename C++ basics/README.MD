This folder is for revising basic c++ concepts and syntax. 

Concepts covered here will be:

How to write a simple C++ program.

Arithmetic, Relational, Logical.

Assignment (=, +=, -=, etc.).

Increment/Decrement (++, --).

If-Else (nested if, if-else ladder).

Switch-case (with break, default).

Loops:

for loop.

do-while loop.

Nested loops.

break & continue.

Abstract class (with pure virtual function).
